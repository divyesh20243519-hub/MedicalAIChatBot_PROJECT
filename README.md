# Medical Vision AI Assistant

This project is a web application built with FastAPI that allows users to upload medical images and ask questions about them in natural language. The application sends the image and query to two Llama 4 vision-language models through the Groq API and displays both responses, allowing users to compare the generated answers.

The project is intended for simple educational and informational medical queries. It is **not** a substitute for professional medical advice, diagnosis, or treatment.

---

## Features

* Upload a medical image
* Ask questions in natural language
* Receive responses from both Llama 4 Scout and Llama 4 Maverick
* Compare answers from both models
* Validate uploaded images before processing
* Handle invalid inputs and API errors gracefully

---

## Technologies Used

* Python
* FastAPI
* Jinja2
* Uvicorn
* Pillow
* Requests
* python-dotenv
* Groq API

---

## How It Works

1. The user uploads a medical image and enters a question.
2. The image is validated using Pillow.
3. The image is encoded into Base64 format.
4. The image and prompt are sent to both Llama 4 vision models through the Groq API.
5. Responses from both models are returned to the frontend.
6. The user can compare the generated answers on the webpage.

---

## Notes

This project was built to explore the use of vision-language models in answering simple questions about medical images while learning FastAPI, API integration, image processing, and backend development. The responses generated by the models are intended for educational purposes only and should not be relied upon for medical decisions.
