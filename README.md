# Vision Language Model Comparison

This project is a simple web application built using FastAPI that lets users upload an image, ask a question about it, and compare the responses generated by two different Llama 4 vision models available through the Groq API.

The purpose of this project was to learn how vision-language models work, how to build APIs with FastAPI, and how to connect a frontend to an AI backend.

---

## Features

* Upload an image from your computer
* Ask any question about the image
* Get responses from both Llama 4 Scout and Llama 4 Maverick
* Compare the outputs of the two models
* Basic image validation before sending requests
* Error handling for invalid images and API failures

## Installation

Clone the repository.

```bash
git clone https://github.com/yourusername/vision-language-comparison.git
cd vision-language-comparison
```

Create a virtual environment (optional but recommended).

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux/macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

Install the required packages.

```bash
pip install -r requirements.txt
```

## How it Works

1. The user uploads an image and enters a prompt.
2. The image is read and converted to Base64.
3. Pillow checks whether the uploaded file is a valid image.
4. The image and prompt are sent to both Llama models through the Groq API.
5. The responses are returned as JSON.
6. The frontend displays both answers side by side.

---

## API Endpoint

### POST `/upload_and_query`

**Form Data**

| Parameter | Type   |
| --------- | ------ |
| image     | File   |
| query     | String |

Example response:

```json
{
    "LLAMA_4SCOUT": "...",
    "LLAMA_4MAVERICK": "..."
}
```
